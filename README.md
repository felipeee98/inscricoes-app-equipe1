## Introdução
Este projeto PHP tem como objetivo criar um sistema dedicado a preenchimento de dados para Editais de Processo do sistema de ensino _Instituto Federal Catarinense_, com Anexos PDF e demais dados pessoais dos incritos. 

# Escopo 
## Auth 
- Seleção de Perfil
- Login e Cadastro

## Modulo do Candidato
- Mural de editais
- Inscrição
- Meu Perfil
- Minhas Incrições

## Painel Administrativo
- Cadastro e gestão de editais
- Visualização de inscrições
- Análise de documentos anexados
- Alteração de status de candidaturas.

## Objetivos

- [ ] Cadastro de candidato e administrador.
- [ ] Cadastro e gerenciamento de editais.
- [ ] Gerenciador de candidaturas com busca e filtros.
- [ ] Acesso aos documentos e dados dos candidatos.
- [ ] Página de inscrição com upload de arquivos.
- [ ] Histórico de status da inscrição.
- [ ] Visualização dos documentos enviados.
- [ ] Cadastro e alteração dos dados pessoais.

# Ferramentas

- Linguagem: PHP
- Framework: Laravel (com Eloquent ORM/SQL built-in)
- Banco de Dados: MariaDB
- Frontend: Bootstrap, HTML5, CSS
- Ambiente: XAMPP
- Versionamento: GitHub
- Suporte ao Desenvolvimento: IA's LLM, Self-hosted models e documentação técnica.
